About Embedded Programming Team

The Embedded Coding team is responsible for the development of firmware and control software for the various robots made by the team. We decide how the robots move and function within the given constraints. Before starting work on the robot, we analyze the theme and work out the next step, which is choosing various sensors and testing them. Members of the team come up with various ideas during this phase. Then, Testing of various sensors and control algorithms is done on an initial prototype of the robot. Following the assembly and wiring of the robot, We develop code-blocks for communication between various microcontrollers present on the robot responsible for various mechanisms.

We aim to keep human error at bay and make the robots autonomous upto a certain level. various algorithms and filters are used to achieve the required level of automation. We are also responsible for the code of the custom Controller developed by the electronics team. We decide which button does what function and how they are inferred by the robot.
